## ❓ Fans 전용 (Fans 역할 필요)

### 릴리즈

1. [릴리즈 페이지](https://github.com/Cotton-Buds/colorpicker/releases)로 이동합니다.
2. `F` 릴리즈를 다운로드합니다. ([F8](https://github.com/Cotton-Buds/colorpicker/releases/tag/F8) 등)

### Usage

1. `injector.exe` 실행합니다.
2. `GenshinImpact.exe` 또는 `YuanShen.exe` 선택 (`cfg.ini`를 확인하여 인젝터가 올바른 경로로 선택되었는지 확인합니다.)
3. [우리 디스코드](https://discord.gg/cottonbuds)에 가서 `micah-bot-verify` 채널에 `/getkey`를 입력하여 인증키를 얻습니다.

<a href="#"><img width="300" height="200" src="https://cdn.discordapp.com/attachments/1126893908597669989/1128329159559622676/image.png"></a>
<a href="#"><img width="365" height="164" src="https://media.discordapp.net/attachments/1126893908597669989/1144891137224613969/image.png"></a>

4. 인젝터에 인증키가 필요하면 인증키를 복사하고 입력합니다.
5. 게임은 자동으로 시작됩니다. 인터페이스가 나타날 때까지 기다리세요
6. `TAB`키를 눌러 [Korepi](https://github.com/Korepi/Korepi) GUI를 엽니다.

## ❓ MICAH의 빌드 설명

### 릴리즈

1. [릴리즈 페이지](https://github.com/Cotton-Buds/calculator/releases)로 이동합니다.
2. `v0.x.x` 릴리즈를 다운로드 합니다. ([v0.9.7](https://github.com/Cotton-Buds/calculator/releases/tag/v0.9.7) 등)

### 사용법

1. `v0.x.x.exe`을 실행합니다.
2. [우리 디스코드](https://discord.gg/cottonbuds)에 가서 `micah-bot-verify` 채널에 `/verify`를 입력하고 **콘솔**에 표시되는 `HWID`를 붙여넣어 **인증키 파일**을 얻습니다.

<a href="#"><img width="965" height="415" src="https://media.discordapp.net/attachments/1126893908597669989/1138425541838516294/Untitled.png"></a>
<a href="#"><img width="1324" height="320" src="https://media.discordapp.net/attachments/1126893908597669989/1138426358217842858/Untitled.png"></a>

3. **인증키 파일**을 다운로드하여 폴더에 너혹 `enc.json` 파일이 `v0.x.x.exe`와 같은 폴더에 있는지 확인하시오
4. `v0.x.x.exe`을 다시 실행합니다.
5. `GenshinImpact.exe` 또는 `YuanShen.exe` 선택 (`cfg.ini`를 확인하여 인젝터가 올바른 경로로 선택되었는지 확인합니다.)
6. 게임은 자동으로 시작됩니다. 인터페이스가 나타날 때까지 기다리세요
7. `TAB`키를 눌러 [Korepi](https://github.com/Korepi/Korepi) GUI를 엽니다.

<a href="#"><img width="270" height="200" src="https://images.drivereasy.com/wp-content/uploads/2018/09/img_5ba9fcbbcb694.png"></a>

## ❓ 일반 유저 전용 -지금은 업데이트 되지 않음.

### 릴리즈

1. [릴리즈 페이지](https://github.com/Korepi/keyauth-cpp-library/releases)에 갑니다.
2. `P` 릴리즈를 다운로드 합니다. (ex. [P5](https://github.com/Korepi/keyauth-cpp-library/releases/tag/P5))

### Usage

1. `telemetry.dll`이 `injector.exe`와 같은 폴더에 있는지 확인합니다.
2. `injector.exe`를 실행합니다.
3. `GenshinImpact.exe` 또는 `YuanShen.exe` 선택 (`cfg.ini`를 확인하여 인젝터가 올바른 경로로 선택되었는지 확인합니다.)
4. 게임은 자동으로 시작됩니다. 인터페이스가 나타날 때까지 기다리세요
5. `TAB`키를 눌러 [Korepi](https://github.com/Korepi/Korepi) GUI를 엽니다.
